关于系统

基于 Spark 的医美数据分析系统面向医美项目销售数据，提供采集与预处理、Parquet/HDFS 存储检查、Spark 聚类与趋势统计，以及本 Web 分析看板。

  • 数据流：CSV → Pandas 清洗（含「统计月份」= 采集日期的年月）→ Parquet/HDFS → Spark 多维 SQL 分析 → JSON 指标；看板通过 /api/analytics 读取 JSON 快照并支持按月份汇总。
  • 技术栈：Python、PySpark、Pandas、Flask、ECharts、JSON。
  • 账号：支持管理员默认账号与注册用户登录；用户数据存储于本地 JSON 文件。

使用左侧导航切换功能模块；各分析页提供「刷新」「导出」等操作。
